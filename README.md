# ControlMyLightsV2


Device: SparkFun Thing Plus - ESP32 WROOM

Pinout: 

* Red 13
* Green 12
* Blue 27
* Ground GND

Libraries:

* #include <ArtnetWifi.h>
* #include <Arduino.h>

How to send it data:

Run Resolume Arena 7. Add new fixture in Fixture Editor (Arena -> Fixture Editor), as 1 pixel in width, 1 pixel height. 
In Output go to Advanced. Click + sign and add DMX Lumiverse and set it to your fixture.
